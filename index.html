ðŸ’° Financial Model: Capital Asset Pricing Model (CAPM)

The CAPM is used to determine the theoretically appropriate required rate of return of an asset.

$$E(R_i) = R_f + \beta_i (E(R_m) - R_f)$$
  • $E(R_i)$: Expected return on the security
  • $R_f$: Risk-free rate
  • $\beta_i$: Beta (systematic risk)
  • $E(R_m)$: Expected return of the market
